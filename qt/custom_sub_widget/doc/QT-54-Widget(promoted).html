一、什么是自定义Widget组件

  • Qt的UI设计器提供了很多GUI设计的界面组件，可以满足常见的界面设计需求。但是某些时候需要设计特殊的界面组件，而在UI设计器的组件面板里没有合适的组件，这时就需要设计自定义的界面组件

二、如何设计自定义的Widget组件

  • 所有组件的基类是QWidget，要设置自定义的界面组件，可以从QWidget继承一个自定义的类，重定义其paintEvent()事件，利用Qt的绘图功能绘制组件外观，并实现需要的其它功能
  • 然后在窗体中放置一个Widget，右击Widget组件，“Protomed to”提升为我们上面定义的那个类（看后面的演示案例）

三、演示案例

  • 设计一个这样的电池组件

第一步：

  • 先创建一个基于QWidget的应用窗体

第二步：

  • 然后再添加一个从QWidget继承的类QmyBattery
  • 在此项目的基础上，点击Qt Creator的“File” ==>"New File or Project" ==>“C++ Class” ==>创建一个基于QWdiget的“qmybattery”类

第三步：

  • 向qmybattery.h文件中书写代码，内容如下
  1. #include <QWidget>
  2. #include <QColor>
  3. #include <QPainter>
  4.  
  5. class qmybattery : public QWidget
  6. {
  7.     Q_OBJECT
  8. private:
  9.     QColor mColorBack=Qt::white;  //背景色
  10.     QColor mColorBorder=Qt::black;//电池边框颜色
  11.     QColor mColorPower=Qt::green; //电量柱颜色
  12.     QColor mColorWarning=Qt::red; //电量短缺时的颜色
  13.     int mPowerLevel=60;           //当前电量（0-100）
  14.     int mWarnLevel=20;            //电量低警示阀值
  15. protected:
  16.     void paintEvent(QPaintEvent *event)Q_DECL_OVERRIDE; //绘图
  17. public:
  18.     explicit qmybattery(QWidget *parent = nullptr);
  19.     void setPowerLevel(int pow); //设置当前电量
  20.     int powerLevel(); //得到当前电量
  21.     void setWarnLevel(int warn); //设置电量低阀值
  22.     int warnLevel(); //得到电量低阀值
  23.     QSize sizeHint(); //返回组件的缺省大小
  24. signals:
  25.     //当电量值改变时发射此信号
  26.     void powerLevelChanged(int);
  27. public slots:
  28. };

第四步：

  • 定义paintEvent()函数
  1. void qmybattery::paintEvent(QPaintEvent *event)
  2. {
  3.     Q_UNUSED(event);
  4.     QPainter painter(this);
  5.     
  6.     //设置QPainter的绘图区
  7.     QRect rect(0,0,width(),height());
  8.     painter.setViewport(rect);
  9.     painter.setWindow(0,0,120,50);
  10.     painter.setRenderHint(QPainter::Antialiasing);
  11.     painter.setRenderHint(QPainter::TextAntialiasing);
  12.     
  13.     //设置画笔
  14.     QPen pen;
  15.     pen.setWidth(2);
  16.     pen.setColor(mColorBorder);
  17.     pen.setStyle(Qt::SolidLine);
  18.     pen.setCapStyle(Qt::FlatCap);
  19.     pen.setJoinStyle(Qt::BevelJoin);
  20.     //设置画刷
  21.     QBrush brush;
  22.     brush.setColor(mColorBack);
  23.     brush.setStyle(Qt::SolidPattern);
  24.     
  25.     painter.setPen(pen);
  26.     painter.setBrush(brush);
  27.     
  28.     //改变rect的区域，绘制电池边框
  29.     rect.setRect(1,1,109,48);
  30.     painter.drawRect(rect);
  31.     
  32.     //改变画刷颜色，改变rect的区域，绘制电池的正极头
  33.     brush.setColor(mColorBorder);
  34.     painter.setBrush(brush);
  35.     rect.setRect(110,15,10,20);
  36.     painter.drawRect(rect);
  37.     
  38.     //画电池柱
  39.     if(mPowerLevel>mWarnLevel) //正常颜色电量柱
  40.     {
  41.         brush.setColor(mColorPower);
  42.         pen.setColor(mColorPower);
  43.     }
  44.     else//电量低时电量柱
  45.     {
  46.         brush.setColor(mColorWarning);
  47.         pen.setColor(mColorWarning);
  48.     }
  49.     painter.setBrush(brush);
  50.     painter.setPen(pen);
  51.     if(mPowerLevel>0)//如果当前有电量，绘制电量柱
  52.     {
  53.         rect.setRect(5,5,mPowerLevel,40);
  54.         painter.drawRect(rect);
  55.     }
  56.     
  57.     //绘制电量百分比文字
  58.     QFontMetrics textSize(this->font());
  59.     QString powerStr=QString::asprintf("%d%%",mPowerLevel);
  60.     QRect textRect=textSize.boundingRect(powerStr); //得到字符串的rect
  61.     painter.setFont(this->font());
  62.     pen.setColor(mColorBorder);
  63.     painter.setPen(pen);
  64.     painter.drawText(55-textRect.width()/2,23+textRect.height()/2,powerStr);;
  65.     
  66. }

第五步：

  • 定义其他成员函数
  1. //设置当前电量
  2. void qmybattery::setPowerLevel(int pow)
  3. {
  4.     mPowerLevel=pow;
  5.     emit powerLevelChanged(pow);
  6.     repaint();
  7. }
  8.  
  9. //得到当前电量
  10. int qmybattery::powerLevel()
  11. {
  12.     return mPowerLevel;
  13. }
  1. //设置电量低阀值
  2. void qmybattery::setWarnLevel(int warn)
  3. {
  4.     mWarnLevel=warn;
  5.     repaint();
  6. }
  7.  
  8. //得到电量低阀值
  9. int qmybattery::warnLevel()
  10. {
  11.     return mWarnLevel;
  12. }
  1. 返回组件尺寸大小
  2. QSize qmybattery::sizeHint()
  3. {
  4.     int H=this->height();
  5.     int W=H*12/5;
  6.     QSize size(W,H);
  7.     return size;
  8. }

第六步：

使用“提升法”，将我们自定义的类对象提升为UI组件

  • ①在UI界面中，移动一个Widget组件到主窗体中，然后右击组件，点击“Promote to”菜单项

  • ②点击之后，出现下面的UI画框

  • ③我们将“Base class name”改为“QWidget”。将Promoted class name改为“qmybattery”。勾选一些选项，然后点击Promote（如果我们有其他的自定义类也可以添加到Promoted Classes中，以便我们选择使用）

  • ④可以看到这个组件的类名称变为了qmybattery

  • ⑤右击这个组件，将其object name改为“battery”

第七步：

  • 在主窗体中再放置一个Horizontal Slider组件和一个QLabel组件，然后与battery这个组件进行排版

第八步：

  • 我们书写Horizontal Slider组件的valueChanged信号，然后调用setPowerLevel函数，这样的话，我们的qmybattery类就会接收到powerLevelChanged信号了

  1. void Widget::on_horizontalSlider_valueChanged(int value)
  2. {
  3.     ui-> battery->setPowerLevel(value);
  4.     QString str=QStringLiteral("当前电量：")+QString::asprintf("%d %%",value);
  5.     ui->label->setText(str);
  6. }
